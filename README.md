# Magni Development Organizational Defaults

This repository (`.github`) contains the default community health files and the organizational profile README for [Magni Development](https://github.com/magnidev). 

## Structure

- `profile/README.md`: The organizational profile displayed at the root of the Magni Development GitHub organization.
- `CODE_OF_CONDUCT.md`: Default code of conduct outlining community expectations across our open-source repositories.
- `SECURITY.md`: Security policy and vulnerability reporting guidelines.

## About Magni Development

Software automation to solve real-world problems and drive business value. Learn more at [magni.dev](https://magni.dev).
